# Documentação do Frontend -- AgroGestão

## Visão Geral

Interface desenvolvida para o sistema AgroGestão, focada em
gerenciamento de produtores, produtos, clientes, pedidos e pagamentos.
Estrutura simples, direta e organizada em layout vertical com navegação
lateral fixa.

## Tela de Login

-   Formulário centralizado.
-   Campos: Usuário e Senha.
-   Botão de entrada.
-   Informação de credenciais padrão exibida abaixo do formulário.

## Dashboard

Após autenticação, o usuário acessa o painel principal contendo: -
Resumo quantitativo de: - Produtores - Produtos - Clientes - Pedidos -
Seções reservadas para gráficos: - Vendas por mês - Produtos mais
vendidos

## Menu Lateral

Itens apresentados: - Dashboard - Produtores - Produtos - Clientes -
Pedidos - Pagamentos - Relatórios - Sair

Menu fixo com destaque para o item ativo.

## Layout

-   Paleta: tons de verde.
-   Componentes com bordas arredondadas.
-   Estrutura responsiva.
